# Nature / IEEE Cover Style Study

These are source links and style notes used to guide the redesigned promotional figure. Do not copy the cover art into this repository.

## Source Links

| Source | Link | Useful Visual Pattern |
|---|---|---|
| Nature cover archive | https://www.nature.com/nature/volumes | Editorial covers use one memorable object and strong light discipline. |
| Nature Machine Intelligence archive | https://www.nature.com/natmachintell/volumes | AI systems shown as networks, fields, particles, circuits, and layered abstractions. |
| Nature Biomedical Engineering archive | https://www.nature.com/natbiomedeng/volumes | Biomedical interfaces, implants, tissue, and computational layers rendered with restrained drama. |
| Scientific Reports article pages | https://www.nature.com/srep/ | Multi-panel figures are source evidence; cover art should compress the mechanism. |
| IEEE Transactions on Biomedical Engineering current issue | https://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=10 | Biomedical signal-processing and device-system clarity. |
| IEEE Journal of Biomedical and Health Informatics current issue | https://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=6221020 | Health AI workflows, reports, and interpretable model outputs. |

## Extracted Style Rules

| Rule | Application In New Cover |
|---|---|
| One central evidence object | A luminous EEG evidence cortex anchors the image. |
| Show interpretation as an overlay | Channel attribution, SHAP bars, and prototype windows sit on top of signal layers. |
| Distinguish model score from evidence | Benchmark lanes and report tiles are separate from the brain/electrode field. |
| Use clinical restraint | Deep background, cyan signal, magenta attribution, amber evidence. |
| Avoid literal diagnosis | The final surface is an interpretable report, not a diagnosis stamp. |

## Design Contract

The redesigned cover should read as:

`EEG windows -> benchmark split -> model attribution -> interpretable report`

It should feel like a biomedical AI cover or graphical abstract, with enough technical specificity to be credible.
