# Roadmap

## Month 1: Public Prototype

- Build synthetic EEG benchmark pipeline.
- Implement a leakage-aware split abstraction.
- Provide PSD + classical classifier baseline.
- Generate metrics, confusion matrix placeholders, and attribution summaries.
- Produce a Markdown report for each run.

## Month 2: Public Dataset Integration

- Add one seizure detection dataset loader.
- Add one sleep staging dataset loader.
- Integrate MNE preprocessing.
- Add EEGNet or shallow ConvNet baseline.

## Month 3: XAI Expansion

- Add saliency and perturbation attribution.
- Add channel and frequency-band stability analysis.
- Add error-case viewer.
- Prepare a methods/demo manuscript.

## Long-Term

- Multi-disease benchmark suite.
- Model cards and dataset cards.
- External validation support.
- Clinical collaborator workflow.
- Interactive dashboard for interpretable EEG reports.
